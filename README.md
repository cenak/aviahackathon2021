# aviahackathon2021

### Сторона клиента
Сторона клиента подразумевает собой рабочие пространства для **человека под самолетом**, **авиакомпании** и **аэропорта**.
Клиент написан на ReactJS

Функции:
- Просмотр заявок
- Создание заявок
- Аналитика услуг

Предлагаемые улучшения:
- Связать с сервером
